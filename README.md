## Carrinho
A small project to mess around with react/redux, the firebase REST api, and a few other things.  Started with the help
create-react-app


### Installation
- pull down the repo (fork/clone)
- yarn install
- yarn start (to run dev - uses port 3000 by default)


### Tests
test scripts:
- yarn test:watch
- yarn test
- yarn test:coverage
